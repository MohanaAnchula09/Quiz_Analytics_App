# InsightBoard – Quiz Analytics Dashboard 

An interactive data analytics dashboard built using React that allows users to take quizzes and visualize performance using dynamic charts.

---

## Live Demo
https://mohanaanchula09.github.io/insightboard-dashboard/

## GitHub Repository
https://github.com/MohanaAnchula09/Quiz_Analytics_App.git

---

## Features

- Multi-page application using React Router
- Interactive charts (Line, Bar, Pie)
- Dynamic filtering and real-time updates
- Persistent data using localStorage
- Responsive UI (mobile + desktop)
- Sidebar navigation

---

## Tech Stack

- React.js
- JavaScript (ES6)
- HTML5 & CSS3
- Recharts
- React Router

---

## Functionality

- Take quiz and track scores
- View results instantly
- Analyze performance trends
- Category-wise insights
